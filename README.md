# University Management System
A console-based University Management System developed in C++ using Object-Oriented Programming (OOP) principles and File Handling.

## Features
* Add new student records
* Search students using Roll Number
* Update existing student information
* Store data permanently using text files
* Simple menu-driven user interface

## Concepts Used
* Classes and Objects
* Encapsulation
* Constructors
* File Handling (`ifstream`, `ofstream`)
* String Manipulation
* Functions and Modular Programming

## Technologies
* C++
* Object-Oriented Programming
* File Management

This project was built as a practice application to strengthen my understanding of C++ OOP concepts and file handling by implementing a real-world student record management system.
